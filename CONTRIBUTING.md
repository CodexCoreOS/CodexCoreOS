# Contributing to Codex Core

Thank you for your interest in Codex Core.

This repository is the public organization profile and top-level GitHub entry point for the Codex Core ecosystem.

## Purpose of this repository

Use this repository for:

- organization profile content
- public-facing GitHub positioning
- links to flagship repositories
- lightweight community and contribution guidance

Do not use this repository as a general implementation codebase.

## Good contributions

Examples of good contributions include:

- improving clarity in the profile README
- refining public-facing positioning
- adding links to public repositories or documents
- improving repository hygiene and standards
- adding issue templates and support guidance

## Avoid

Please avoid:

- speculative claims about unreleased systems
- mixing sensitive implementation details into public-facing copy
- broad branding changes that break ecosystem clarity
- using this repo as a catch-all workspace

## Pull request standard

Each pull request should explain:

- what changed
- why it improves the public GitHub entry point
- whether it changes positioning or only repository hygiene

When in doubt, prefer the narrower and clearer change.
